title: '一分钟，搭建ESP32 Rust开发环境'
---

# 注意

与`Cortex-M`系列单片机开发不同，`ESP32`的工具链稍显特殊，但是都属于一键式安装，遵循以下命令安装即可。

---

# 按照顺序，执行以下命令

```bash
# 根据模板生成工程
cargo install esp-generate
# esp32开发包安装工具
cargo binstall espup
# 安装esp32开发包
espup install
# 安装烧录工具
cargo install espflash
```

完成以上步骤之后，`ESP32`的开发环境就搭建完毕了。恭喜，你的技术栈又丰富了许多。
