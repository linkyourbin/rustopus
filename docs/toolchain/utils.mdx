## 下载/调试工具
配合`cargo`可以一键下载运行程序，也可以单独使用，用于烧录并复位芯片。

```bash
cargo install probe-rs-tools
```
### 单独使用`probe-rs`

```bash
# 将xxx.elf固件烧录到目标芯片
probe-rs download xxx.elf --chip STM32F405RG
# 复位目标芯片
probe-rs reset --chip STM32F405RG
```
## 零碎工具

```bash
cargo install cargo-binutils
rustup component add llvm-tools
```
